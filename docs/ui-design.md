# AI Studio 工作台 - UI 设计方案

> 生成日期: 2026-06-28
> 版本: v1.0

## 一、设计系统

| 属性 | 选择 | 说明 |
|------|------|------|
| **风格** | Modern Glassmorphism + Dark Mode | 参考 Cursor/Dify 的专业效率工具风格 |
| **主色** | `#6366F1` (Indigo) | 现代感强，适合 AI 工具 |
| **强调色** | `#10B981` (Emerald) | 成功/发送按钮 |
| **背景** | `#0F172A` (Slate-900) | 深色主题，减少视觉疲劳 |
| **文字** | `#F8FAFC` (Slate-50) | 高对比度，易读 |
| **字体** | Plus Jakarta Sans | 现代、专业、可读性好 |

### 配色方案

```
深色主题配色:

背景层级:
  █ bg-slate-950 (#020617) - 最深（页面背景）
  █ bg-slate-900 (#0F172A) - 深（侧边栏、卡片）
  █ bg-slate-800 (#1E293B) - 中（输入框、hover态）
  █ bg-slate-700 (#334155) - 浅（分割线、次级区域）

功能色:
  █ primary: #6366F1 (Indigo-500) - 主按钮、选中态
  █ success: #10B981 (Emerald-500) - 成功、发送
  █ warning: #F59E0B (Amber-500) - 警告、重要提示
  █ error: #EF4444 (Red-500) - 错误
  █ accent: #8B5CF6 (Violet-500) - AI 相关元素
```

## 八、组件层级与状态

### 8.1 状态设计

| 组件 | 状态 | 视觉表现 |
|------|------|----------|
| **按钮** | Default / Hover / Active / Disabled / Loading | 颜色变化 + 微缩放 |
| **输入框** | Default / Focus / Error / Disabled | 边框高亮 + 阴影 |
| **卡片** | Default / Hover / Selected | 背景色 + 边框 + 阴影 |
| **模型选择** | Connected / Disconnected / Loading | 绿点/灰点/黄点指示器 |
| **消息气泡** | Thinking / Streaming / Complete / Error | 打字动画 / 静态 / 错误提示 |

| 场景 | 时长 | 效果 |
|------|------|------|
| 消息出现 | 300ms | fade-in + slide-up |
| 流式打字 | 无限制 | 逐字符显示 ▌光标闪烁 |
| 按钮反馈 | 150ms | scale(0.98) → scale(1) |
| 页面切换 | 200ms | fade |
| 加载骨架 | 1500ms | animate-pulse |
